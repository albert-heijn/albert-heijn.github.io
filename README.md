## Development

To set up your environment to develop this theme, run `bundle install`.

To test the theme, run `bundle exec jekyll serve` and open your browser at `http://localhost:4000/`. This starts a Jekyll server using content in the `/` directory. As modifications are made to the theme and test site, it will regenerate and you should see the changes in the browser after a refresh.
